android error.

  1. Chrome のアドレスバー（URL表示窓）の右側にある 縦3点リーダー をタップします。
  2. [設定] をタップします
  3. [サイトの設定] をタップします
  4. [カメラ] をタップし、[オン] をタップします
    このページの”ブロック中”に以下のURLがある場合、URLをタップし [許可] をタップします。

ios error.

  1. 設定アプリから[Safari] -> [カメラ]
  2. 「確認」もしくは「許可」にしてください。
  1. アドレスバーの左にある「あぁ」をクリック -> [Webサイトの設定]
  2. [カメラ]->「確認」か「許可」にしてください。

pc error.

PC Chrome

  1. アドレスバー左の「鍵アイコン」クリック。その後「サイトの設定」をクリックします。
  2. 「権限 > カメラ」項目をクリックし、「許可」を選択してブラウザのリロードを行ってください。

PC Safari

  1. ツールバーから「Safari」> 「環境設定」をクリックします。
  2. 「Webサイト > カメラ」から「現在開いているWebサイト」を確認し、本人確認を行うサイトのアクセスを「許可」にしてください。
  3. ブラウザのリロードを行ってください。
